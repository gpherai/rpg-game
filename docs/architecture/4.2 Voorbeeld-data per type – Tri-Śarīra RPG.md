# 4.2 Voorbeeld-data per type – Tri-Śarīra RPG

**Versie:** 0.1  
**Auteur:** Gerald (met Game Designer- & Architect-ondersteuning)  
**Status:** Design-voorbeeld. Niet definitief; bedoeld als startpunt voor vertical slice + schema-validatie.

Doel: **concrete voorbeeldbestanden** schetsen voor de belangrijkste data-types, zodat Claude/Codex hieruit echte `*_example.json`-files kan genereren en de systemen in de vertical slice al iets tastbaars hebben.

Vertical slice-focus: **R1 / Chandrapur + eerste route + eerste shrine**, MC + 1 companion, enkele enemies, items, skills, quests, dialogue.
